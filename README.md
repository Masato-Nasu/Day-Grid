# DayGrid — 一日を正方格子に積む時計ウィジェット

![DayGrid Android Widget](Screenshot1.png)

横60（分）×縦24（時）の正方格子です。過ぎた1分が白い粒として積もり、現在の一粒だけが1秒周期で点滅します。0:00になると、淡い1,440粒だけの一日の初期状態へ戻ります。

## インストール
配布APKをAndroid端末へダウンロードして開き、インストールします。既存版と署名が異なる場合は、旧版をアンインストールしてから入れてください。インストール後、ホーム画面を長押し → ウィジェット → DayGrid を配置します。

## ソースからのビルド
Android Studioで開いてRun、または：

    ./gradlew assembleDebug
